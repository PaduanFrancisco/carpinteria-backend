# Carpintería Paduán - Backend

API REST completa para gestión de carpintería.

**Deploy en producción**  
https://carpinteria-backend.vercel.app

### Rutas principales
- GET  `/api/productos` → Lista todos los productos  
- GET  `/api/clientes`  
- POST `/api/pedidos`  
- GET  `/api/categorias`

### Tecnologías
- Node.js + Express
- TypeScript
- MySQL
- Deploy: Vercel

### Estado
100 % funcional y online 24/7
